# Product Overview

**Socialwise Chatwit** is a comprehensive AI-powered customer service platform specialized in social media automation and legal support for lawyers. Built with Next.js 15, React, TypeScript and Prisma, it integrates OpenAI (GPT-4, GPT-4o, DALL-E, Whisper) and Meta/WhatsApp Business services, using PostgreSQL for persistence. The platform serves micro, small, and medium enterprises with complete digital management and automated customer service solutions.

## Core Features

- **ChatWit AI Assistant**: Complete OpenAI integration (GPT-4, GPT-4o, DALL-E, Whisper) with customizable system prompts
- **Social Media Automation**: Instagram Business automation, automated responses, lead collection
- **Legal System**: OAB lead management, document processing, proof analysis, automated correction mirrors
- **WhatsApp Business Integration**: Official Meta Business API integration with dynamic templates
- **Advanced Admin Panel**: Real-time dashboard, user management, detailed reports, SSE monitoring

## Target Users

- **Lawyers**: Complete client management, automated proof correction, legal specialization support
- **Businesses**: 24/7 automated customer service, lead generation, sales automation
- **Agencies**: Multi-client management, white-label solutions, scalable automation

## Key Value Propositions

- AI-powered automation reducing manual work by 80%
- Specialized legal document processing and analysis
- Multi-platform social media management
- Real-time monitoring and analytics
- Scalable architecture supporting thousands of simultaneous interactions
